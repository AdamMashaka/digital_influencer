Here are some common questions and answers about our system.

General Questions

Technical Questions

The system is compatible with all major web browsers and can be accessed on both desktop and mobile devices.

The system can support an unlimited number of users, depending on the subscription plan.

The system comes with a one-year warranty for all subscription plans.

The system is cloud-based and does not require any installation. You can start using it immediately after signing up.

User Roles

The system supports various user roles including Admin, Organizer, and Attendee. Each role has different permissions and access levels.

User roles can be changed by an Admin. If you need to change your role, please contact the system administrator.

Organizers can create and manage events, view registrations, and generate certificates for attendees.

Attendees can register for events, view event details, and download their certificates after the event.

Reports Generation

The system allows generating various reports including event attendance, registration details, and certificate issuance.

Reports can be generated from the Reports section in the dashboard. Select the type of report you want to generate and specify the required parameters.

Yes, reports can be exported in various formats including PDF and Excel.

Reports are updated in real-time as data is entered into the system.
